#### Titanic Survival Prediction

Machine Learning and Excel Analytics Project

---

#### Project Overview

This repository contains an end-to-end Titanic Survival Prediction project built using Python and Excel.  
It includes data cleaning, EDA, feature engineering, machine learning modelling, and an Excel dashboard for business insights.

---

#### What’s Inside?

> Each part of this project is organised into clear sections.  
> You can open the notebook for ML, use Excel for analytics, or explore datasets individually.

---

#### Project Breakdown

| Section | Description |
|--------|-------------|
| Data Exploration | Understanding survival patterns |
| Data Cleaning | Handling missing values and preprocessing |
| Feature Engineering | Preparing new and existing features |
| Machine Learning | Random Forest model for prediction |
| Evaluation | Accuracy and feature importance |
| Excel Dashboard | Visual analytics and slicers |
| Kaggle Submission | Final CSV generated from test data |

---

#### Dataset Details

| File | Purpose |
|------|---------|
| train.csv | Kaggle training dataset |
| test.csv | Kaggle test dataset |
| submission.csv | Predictions generated by the model |
| titanic_python_project.ipynb | Full Python ML notebook |
| titanic_excel_dashboard.xlsx | Excel dashboard with charts and slicers |

---

#### Key Insights

| Insight | Observation |
|---------|-------------|
| Gender | Females had higher survival rates |
| Class | First-class passengers survived more |
| Age | Children survived more often than adults |
| Fare | Higher fare increased survival chances |
| Family Size | Certain family sizes improved survival |
